## Açıklama

**Fake Data Generator PRO**, web üzerinde tamamen **frontend (HTML + JavaScript)** çalışan, rastgele sahte veri üreten bir projedir.  
- Üretilen veriler **tamamen uydurma** olup gerçek kişilerle ilişkili değildir.  
- **Gerçek Türk isimleri** kullanılmıştır, fakat diğer bilgiler tamamen sahte ve test amaçlıdır.  
- Siteyi kopyalamak veya dağıtmak **yetkisiz olarak yapılmamalıdır**.  

---

## Özellikler

- Rastgele veri üretimi:
  - İsim & Soyisim
  - Ülke & Şehir
  - Adres
  - Email
  - Telefon
  - IP
  - GPS koordinatları
  - Kullanıcı adı
- **TXT / CSV / JSON** formatında veri indirme
- **Güvenlik kodu (CAPTCHA)** ile veri üretme kontrolü
- Basit ve sade tasarım
- Tamamen **frontend** → sunucu veya PHP gerektirmez

---

## Kullanım

1. `index.html` dosyasını tarayıcıda açın.
2. Güvenlik kodunu girerek doğrulayın.
3. Üretmek istediğiniz veri miktarını girin.
4. **Generate** butonuna tıklayın.
5. İsterseniz **TXT, CSV veya JSON** formatında veriyi indirin.
